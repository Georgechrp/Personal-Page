Chordio

Find and organize guitar chords with ease – built with Jetpack Compose

About the App

This Android application is the final thesis project of an undergraduate student at the University of Piraeus. It is developed using Kotlin and Jetpack Compose, offering a clean UI for managing guitar chords, saving favorites, and discovering new songs.
